# HAB-BIP39

**哈勃未来 · BIP39 助记词生成器**
完全离线、开源、用户可自主管理安全。

---

## 项目简介

HAB-BIP39 是一个轻量级 **BIP39 英文助记词生成器**，用于生成 12 或 24 词比特币助记词。项目理念：

* **安全**：全程离线生成助记词，助记词不上传任何服务器。
* **透明**：开源代码可自检，默认加载官方英文词表，同时显示 SHA256 校验值，用户可验证词表完整性。
* **易用**：支持手动上传词表覆盖默认词表；提供复制、清除和下载源码功能。

适合普通用户、比特币爱好者或开发者使用。

---

## 功能特点

* 支持 **12 词 / 24 词助记词**生成
* 默认加载 `english.txt`，显示 SHA256 校验值
* 支持手动上传词表，用户可自行验证安全性
* 助记词生成完全离线，不联网，保障隐私
* 提供复制助记词和下载 HTML 源码功能

---

## 使用方法

1. 将 `index.html` 和官方 `english.txt` 放在同一目录。
2. 在浏览器中打开 `index.html`（推荐现代浏览器，如 Chrome/Edge/Firefox）。
3. 页面会自动加载默认词表，并显示 SHA256 校验值。
4. 选择生成助记词长度（12 词或 24 词），点击“生成助记词”。
5. 助记词生成后，可复制或手抄保存。

> ⚠️ 强烈建议断网使用，并妥善保存助记词。

### 手动上传词表

* 点击“上传 BIP39 英文词表”，选择官方 `english.txt` 文件，可覆盖默认词表。
* 官方词表来源：[BIP39 英文词表](https://github.com/bitcoin/bips/blob/master/bip-0039/english.txt)
* 页面会显示该文件的 SHA256 校验值，用户可自行验证。

---

## 安全提示

* **完全离线**：确保生成过程中断开网络。
* **验证词表**：使用官方 `english.txt`，对比 SHA256 校验值。
* **妥善保存**：助记词一旦泄露，数字资产安全无法保障。
* **开源透明**：源码可下载并自行审查。

---

## 部署方式

1. **本地使用**：直接双击 `index.html` 打开（部分浏览器可能限制 fetch，推荐本地服务器方式）

   ```bash
   python3 -m http.server 8000
   ```

   然后访问 `http://localhost:8000/index.html`
2. **云端部署**：上传到 GitHub Pages、Cloudflare Pages 或任何静态站点服务即可使用。
