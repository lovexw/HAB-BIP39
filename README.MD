# HAB-BIP39

**哈勃未来 · BIP39 助记词生成器**  
完全离线、开源、用户可自主管理安全。

---

## 项目简介

HAB-BIP39 是一个轻量级 **BIP39 英文助记词生成器**，用于生成 12 或 24 词比特币助记词。项目理念：

* **安全**：全程离线生成助记词，助记词不上传任何服务器。
* **透明**：开源代码可自检，内置官方 BIP39 英文词表（2048词），可随时查看验证。
* **纯净**：无任何外部依赖，无第三方脚本，无远程资源加载。
* **易用**：提供复制、清除、查看词表和下载源码功能。

适合普通用户、比特币爱好者或开发者使用。

---

## 功能特点

* 支持 **12 词 / 24 词助记词**生成
* **内置 BIP39 标准英文词表**（2048 词），无需外部文件
* **查看词表功能**：可查看完整的 2048 个单词，按索引排序，方便对比验证
* **完全离线**：无任何外部依赖或第三方脚本
* **纯净安全**：无 Google Fonts 等外部资源，使用系统字体
* 助记词生成完全基于浏览器的 Web Crypto API，保障隐私
* 提供复制助记词和下载完整 HTML 源码功能

## 使用方法

### 基本使用

1. 在浏览器中打开 `index.html`（推荐现代浏览器，如 Chrome/Edge/Firefox）。
2. **强烈建议断开网络连接**，确保完全离线使用。
3. 选择生成助记词长度（12 词或 24 词），点击"生成助记词"。
4. 助记词生成后，可复制或手抄保存。

> ⚠️ **安全警告**：请务必在**断网环境**中使用本页面，并妥善保存助记词。助记词一旦泄露，数字资产安全无法保障。

### 查看内置词表

* 点击"查看词表"按钮，可以查看完整的 2048 个 BIP39 英文单词
* 每个单词都带有索引编号（0-2047），方便与官方词表对比验证
* 官方词表来源：[BIP39 英文词表](https://github.com/bitcoin/bips/blob/master/bip-0039/english.txt)

### 验证安全性

您可以通过以下方式验证本工具的安全性：

1. **查看内置词表**：点击"查看词表"按钮，将显示的 2048 个单词与官方 BIP39 词表逐一对比
2. **审查源代码**：所有代码都在单个 HTML 文件中，易于审查
3. **检查网络请求**：使用浏览器开发者工具（F12），确认无任何网络请求
4. **离线测试**：断网后使用，确保工具完全离线运行

## 安全提示

* **完全离线**：确保生成过程中断开网络。
* **验证词表**：使用"查看词表"功能，对比官方 BIP39 英文词表。
* **妥善保存**：助记词一旦泄露，数字资产安全无法保障。
* **开源透明**：源码可下载并自行审查，无任何隐藏代码。
* **无外部依赖**：本工具不加载任何外部资源（包括字体、脚本等）。

---

## 技术特性

* **纯 HTML + CSS + JavaScript**：无需构建工具，单文件即可运行
* **Web Crypto API**：使用浏览器原生的 `crypto.getRandomValues()` 生成随机熵
* **BIP39 标准**：严格遵循 BIP39 规范，使用 SHA-256 计算校验和
* **内置词表**：2048 个英文单词直接编码在 JavaScript 中
* **无外部请求**：无任何 CDN、字体、脚本等外部资源依赖

## 部署方式

### 本地使用

直接双击 `index.html` 打开即可使用（推荐方式）。

如果浏览器限制本地文件访问，可以使用本地服务器：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/index.html`

### 云端部署

可上传到 GitHub Pages、Cloudflare Pages 或任何静态站点服务。

**注意**：虽然支持云端部署方便访问，但生成助记词时**必须断网使用**。
